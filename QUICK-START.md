### ‡∑Ä‡∑í‡∂ö‡∂Ω‡∑ä‡∂¥‡∂∫ 3: GitHub Actions (Automatic Build)

‡∂î‡∂∂‡∑ö code ‡∂ë‡∂ö GitHub ‡∂ë‡∂ö‡∂ß push ‡∂ö‡∂ª‡∂± ‡∂î‡∂±‡∑ä‡∂± ‡∑Ä‡∑ô‡∂Ω‡∑è‡∑Ä‡∂ß automatic build ‡∂ë‡∂ö‡∂ö‡∑ä!


‡∂∏‡∑ö‡∂ö `.github/workflows/build.yml` file ‡∂ë‡∂ö‡∂ß paste ‡∂ö‡∂ª‡∂±‡∑ä‡∂±:

```yaml
name: Build APK

on:
  push:
    branches: [ main, master ]
  workflow_dispatch:

jobs:
  build:
    runs-on: ubuntu-latest
    
    steps:
      - uses: actions/checkout@v3
      
      - name: Setup Node.js
        uses: actions/setup-node@v3
        with:
          node-version: 18
      
      - name: Setup Java
        uses: actions/setup-java@v3
        with:
          distribution: 'zulu'
          java-version: 11
      
      - name: Install dependencies
        run: npm install
      
      - name: Build APK
        run: |
          cd android
          chmod +x gradlew
          ./gradlew assembleRelease
      
      - name: Upload APK to Releases
        uses: softprops/action-gh-release@v1
        if: startsWith(github.ref, 'refs/tags/')
        with:
          files: android/app/build/outputs/apk/release/app-release.apk
```

